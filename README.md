# Todo List Python

Application de gestion de tâches réalisée en Python.

## Fonctionnalités

- Ajouter une tâche
- Lister les tâches
- Terminer une tâche
- Supprimer une tâche
- Sauvegarder dans un fichier JSON

## Bonus

- Gestion des priorités
- Date d’échéance

## Lancer le projet

```bash
python main.py
